# Golang知识点

```
  1. 只能与nil能进行比较数据类型
      slice,map,func
      
  2. 不可寻址的变量
      值为struct的map，常量,{}T
      例如：
        case1:
        var m1 = make(map[int]struct{
          name string
        })
        m1[0].name = "haoge"  //×，map[key]struct 中 struct 是不可寻址的，所以无法直接赋值
        
        case2:
        const name = "string" //此时const是可不寻址的
        
        case:
        type T {}
        
        func (t *T)test(){}
        
        {}T.test()            //此时 {}T是不可寻址的
        
        //应该这样使用：
        tt := T{}
        tt.test()
        
  3. := 短变量的注意事项
     3.1 结构体的字段不能使用短变量 stu.name,age := "haoge",27
   
  4. defer
  
  5. 指针不支持索引计算
      type Student struct {
        ls:[]int
      }
      
      func main() {
        stu := new(Student)
        stu.ls[0] = 100   //stu是指针，指针不支持索引计算
      }
      
  
  6. 多重赋值（有先后顺序）
     i,s[i] := num+1,num+2
        6.1 赋值顺序，等号左边计算->等号右边计算->赋值
     
  7. i++
      7.1 go语言不存在 ++i,--i
      7.2 i++,i--是语句，不是表达式，不能进行赋值操作，如 age := i++
      
  8. 变量隐藏
      8.1 由于作用于不同造成了变量隐藏现象
      
  9. 常量
      9.1 iota只能使用在const代码块中的常量定义，用于标定常量块中的常量行索引
      9.2 常量可以定义，但是不适用
      
  10. 定义不适用，不会报错的情况
      全局变量，常量，函数参数
  
  11. cap()函数使用
      cap不能用于map类型
      
  12. slice,map只进行声明就使用的情况
      12.1  var m map[string]string 
            m["one"] = "one"      //错误，需要使用make函数分类内存
            
      12.2  var s []int
            s := append(s,1,2,3)
            //切片支持声明不必初始化的使用，map必须初始化后使用
      
  13. slice数据结构的初始化
      s := make([]type,cap)
      //声明切片时，需要声明其容量
```
